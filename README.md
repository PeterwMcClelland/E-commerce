# E-commerce Back End

# Description
This app is the back end for an e-commerce site. Made by usung Express.js while working with Sequelized and Mysql database.

# Built with 
Javascript,
Mysql,
Sequelized,
Express,
dotenv.

You will also need to create a .env to run the app.
